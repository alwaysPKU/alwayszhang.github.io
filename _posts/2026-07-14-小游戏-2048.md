# 🎮 2048 小游戏

一个经典的 2048 数字合并游戏，纯 HTML + CSS + JavaScript 实现，单文件即可运行。

## 玩法

- **键盘**：方向键控制
- **触屏**：滑动操作
- **目标**：合并相同数字，达到 2048！

## 特色

- 🎨 经典配色方案，每个数字都有独特颜色
- ✨ 流畅的滑动和合并动画
- 📱 完美适配桌面和移动端
- 🏆 本地最高分记录（localStorage）
- 🎯 胜利/失败检测
