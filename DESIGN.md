# DESIGN.md

## 品牌与视觉方向
- 博客名：HalfSugar（半甜不要腻）
- 风格定位：简洁、现代、阅读友好，避免花哨装饰
- 气质：技术博客 + 个人空间，兼具专业感与亲和力

## Design Tokens

### 色彩
- 主色调：深蓝灰（oklch 0.45 0.15 260），沉稳不沉闷
- 背景：暖白（light）/ 深蓝灰（dark），避免纯白纯黑
- 强调色：主色透明度变体，用于标签、分类等

### 字体
- 正文：PingFang SC / Hiragino Sans GB / Microsoft YaHei
- 代码：Consolas / Menlo / Monaco 等等宽字体

## 布局与响应式
- 内容区最大宽度 72ch（阅读最佳宽度）
- 移动端适配：汉堡菜单、弹性网格
- 游戏中心：2/3/4 列自适应网格

## 交互与状态
- 暗色模式：跟随系统 / 手动切换
- 卡片悬停：边框高亮 + 微阴影
- 链接：下划线渐显过渡

## 设计禁忌
- 不使用渐变背景或花哨动效
- 不使用过多颜色，保持克制
- 避免大段纯文字无层次
